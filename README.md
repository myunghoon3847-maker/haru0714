# 하루니혼 Lite v1.0.0

매일 10분씩 일본어 문자, 기초 단어, 생활회화를 학습하는 스마트폰용 웹앱입니다.

## 포함 기능

- 히라가나 46자와 가타카나 46자 발음 학습
- 기초 단어·수량 표현 1,000개
- 상황별 생활회화 100문장
- 일본어 음성 듣기
- 기억 상태에 따른 간격 반복 카드 학습
- 문자·단어·회화 혼합 퀴즈
- 오답 자동 저장과 복습 완료 처리
- 검색, 분류, 즐겨찾기, 암기 표시
- 일일 학습 목표, 연속 학습일, 주간 기록
- 다크 모드, 큰 글자, 자동 발음
- 기기 내부 학습기록 저장
- 홈 화면 설치와 오프라인 캐시

## 스마트폰 설치

HTTPS로 배포된 주소를 Chrome 또는 삼성 인터넷에서 연 뒤 브라우저 메뉴의 **앱 설치** 또는 **홈 화면에 추가**를 선택합니다. iPhone Safari에서는 공유 메뉴의 **홈 화면에 추가**를 선택합니다.

학습 기록은 서버로 전송하지 않고 사용 중인 기기에만 저장됩니다.
